# car-parking-program-by-cpp

This mini project in CPP Car parking system is a console application without graphic developed using the CPP programming language.

# FEATURES:
  - Park Rickshaw
  - Park Car 
  - Park Bus
  - Show the Record
  - Delete the Record
 
 # TOOLS:
  - Object Oriented Programming language CPP
  - Code::Blocks IDE
